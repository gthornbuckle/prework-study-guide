HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The h1 - h6 elements represent the level of heading a text block represents.
  • The p element represents a block of text.
  • The ul, ol and li elements represent unordered lists, order lists and list items respectively.
  • The img element contains information about images displayed on the webpage.
  • The br element creates a line break between two blocks of content.
  • The a element creates links to the same webpage or other webpages.

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • CSS rules can be used to define properties of an element including width, height and colour
  • A period preceding a CSS rule designates a class selector
  • Any HTML element assigned to this class will gain the properties assigned to it
  • CSS colours can be assigned using semantic references or hexdecimal values

Git

  • git status: Checks current working branch.
  • git checkout -b branch-name: Creates a new branch and switches to it
  • git add -A: Stages changes to save to github
  • git commit -m "message": Commits changes locally with message detailing them
  • git pull origin main: Pulls base branch to current feature branch
  • git push origin branchname: Pushes changes to remote repository

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An if statement will be executed if the provided condition is truthy
  • else if allows an if statement to have multiple conditions
  • An array is a single variable that holds a group of data
  • Arrays can gold multiple different types of data such at strings, integers and booleans
  • A for loop will execute code over and over until interupted by another conditional statement
  • for loops are extremely useful for DRY coding
